快速开始

QUICK START

1. Fork 项目

点击 GitHub 右上角的 Fork 按钮，fork 本仓库到你的账户

2. 修改配置

在仓库中编辑 ham/data.json 文件，填写你的个人信息

  • 呼号 (callsign)
  • 姓名 (name)
  • 操作等级、 CQ/ITU 分区
  • 设备信息等

3. 启用 GitHub Pages

在仓库设置中启用 GitHub Pages

  • 进入仓库 → Settings → Pages
  • Source 选择 "main branch"
  • 保存后访问 username.github.io/Ham_Hero/ham

配置文件说明

CONFIGURATION

data.json

编辑 ham 文件夹下的 data.json 文件来配置你的个人信息

{
  "callsign": "BG7LZQ",             // 呼号
  "name": "Darion",               // 姓名
  "operatorLevel": "A级",         // 操作员等级
  "cqZone": 24,                 // CQ分区
  "ituZone": 44,                // ITU分区
  "grid": "OL51",               // 网格坐标
  "location": {
    "country": "中国",
    "province": "广东",
    "city": "茂名"
  },
  "description": "A级业余无线电操作员",
  "equipment": {
    "handheld": ["设备1", "设备2"],
    "antenna": ["天线1", "天线2"]
  }
}

注意事项

NOTICE

背景视频

项目使用 bg.mp4 作为背景视频，你需要：

  • 将你的背景视频重命名为 bg.mp4
  • 放入 Ham_Hero 根目录（与 index.html 同级）
  • 建议使用静音、循环的 MP4 视频文件
